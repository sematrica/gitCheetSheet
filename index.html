Git Cheat Sheet

This page displays all Git commands, categorized, with brief descriptions and example usage. Each example is in a black box (just like in a terminal) and can be copied directly.

Main Porcelain Commands

git add
Add file contents to the index (staging area).
git add [file1] [file2]
git am
Apply a series of patches from a mailbox.
git am < mailbox.patch
git archive
Create an archive of files from a named tree.
git archive --format=tar HEAD > source.tar
git bisect
Use binary search to find the commit that introduced a bug.
git bisect start git bisect bad git bisect good v2.0
git branch
List, create, or delete branches.
git branch # List local branches git branch new-branch git branch -d old-branch
git bundle
Move objects and refs by archive.
git bundle create repo.bundle HEAD master git bundle verify repo.bundle
git checkout
Switch branches or restore working tree files.
git checkout main git checkout -- [file]
git cherry-pick
Apply changes introduced by existing commits.
git cherry-pick [commit-hash]
git citool
Graphical alternative to git commit.
git citool
git clean
Remove untracked files from the working tree.
git clean -fd
git clone
Clone a repository into a new directory.
git clone [url] [directory]
git commit
Record changes to the repository.
git commit -m "Commit message" git commit --amend
git describe
Give an object a human-readable name based on an available ref.
git describe --tags
git diff
Show changes between commits, commit and working tree, etc.
git diff git diff HEAD~1 HEAD
git fetch
Download objects and refs from another repository.
git fetch [remote]
git format-patch
Prepare patches for email submission.
git format-patch -n [base_commit]
git gc
Cleanup unnecessary files and optimize the local repository.
git gc --prune=now --aggressive
git gitk
Launch the Git repository browser (graphical).
gitk
git grep
Print lines matching a pattern.
git grep "functionName"
git gui
A portable graphical interface to Git.
git gui
git init
Create an empty Git repository or reinitialize an existing one.
git init
git log
Show commit logs.
git log git log --oneline --graph
git maintenance
Run tasks to optimize Git repository data.
git maintenance run
git merge
Join two or more development histories together.
git merge [branch]
git mv
Move or rename a file, directory, or symlink.
git mv old_filename new_filename
git notes
Add or inspect object notes.
git notes add -m "Note text" [commit] git notes show [commit]
git pull
Fetch from and integrate with another repository or a local branch.
git pull [remote] [branch]
git push
Update remote refs along with associated objects.
git push [remote] [branch]
git range-diff
Compare two commit ranges (e.g. two versions of a branch).
git range-diff upstream..topic upstream..topic-v2
git rebase
Reapply commits on top of another base tip.
git rebase main git rebase -i HEAD~3
git reset
Reset current HEAD to the specified state.
git reset [commit] git reset --hard HEAD~1
git restore
Restore working tree files.
git restore [file] git restore --staged [file]
git revert
Revert some existing commits.
git revert [commit]
git rm
Remove files from the working tree and the index.
git rm [file]
git scalar
A tool for managing large Git repositories (incorporated in Git).
git scalar register .
git shortlog
Summarize git log output.
git shortlog -n -s
git show
Show various types of objects (commits, tags, etc.).
git show [commit]
git sparse-checkout
Reduce your working tree to a subset of tracked files.
git sparse-checkout init git sparse-checkout set folder/subfolder
git stash
Stash the changes in a dirty working directory away.
git stash git stash pop
git status
Show the working tree status.
git status
git submodule
Initialize, update, or inspect submodules.
git submodule add [url] git submodule update --init --recursive
git switch
Switch branches.
git switch [branch]
git tag
Create, list, delete or verify a tag object signed with GPG.
git tag v1.0 git tag -d v1.0
git worktree
Manage multiple working trees.
git worktree add ../path [branch] git worktree remove ../path

Ancillary Commands / Manipulators

git config
Get and set repository or global options.
git config --global user.name "John Doe"
git fast-export
Git data exporter.
git fast-export HEAD > repo.export
git fast-import
Backend for fast Git data importers.
git fast-import < repo.export
git filter-branch
Rewrite branches (historical rewriting).
git filter-branch --tree-filter 'rm filename' HEAD
git mergetool
Run merge conflict resolution tools.
git mergetool
git pack-refs
Pack heads and tags for efficient repository access.
git pack-refs --all
git prune
Prune all unreachable objects from the object database.
git prune
git reflog
Manage reflog information.
git reflog
git remote
Manage set of tracked repositories.
git remote -v git remote add origin [url]
git repack
Pack unpacked objects in a repository.
git repack -ad
git replace
Create, list, or delete refs to replace objects.
git replace [object] [replacement] git replace -l

Ancillary Commands / Interrogators

git annotate
Annotate file lines with commit information.
git annotate [file]
git blame
Show what revision and author last modified each line.
git blame [file]
git bugreport
Collect information for filing a bug report.
git bugreport
git count-objects
Count unpacked objects and their disk consumption.
git count-objects -v
git diagnose
Generate a zip archive of diagnostic information.
git diagnose
git difftool
Show changes using common diff tools.
git difftool HEAD~1 HEAD
git fsck
Verify connectivity and validity of objects in the database.
git fsck --full
git gitweb
Git web interface (web frontend to Git repositories).
git instaweb --httpd=webrick
git help
Display help information about Git.
git help [command]
git instaweb
Instantly browse your working repo in gitweb.
git instaweb --httpd=webrick
git merge-tree
Perform merge without touching index or working tree.
git merge-tree [base] [head1] [head2]
git rerere
Reuse recorded resolution of conflicted merges.
git rerere
git show-branch
Show branches and their commits.
git show-branch
git verify-commit
Check the GPG signature of commits.
git verify-commit [commit]
git verify-tag
Check the GPG signature of tags.
git verify-tag [tag]
git version
Display version information about Git.
git version
git whatchanged
Show logs with differences each commit introduces.
git whatchanged

Interacting with Others

git archimport
Import a GNU Arch repository into Git.
git archimport [archive]
git cvsexportcommit
Export a single commit to a CVS checkout.
git cvsexportcommit -c [commit]
git cvsimport
Salvage your data out of a CVS repository.
git cvsimport -d [cvsroot] [module]
git cvsserver
A CVS server emulator for Git.
git cvsserver
git imap-send
Send a collection of patches from stdin to an IMAP folder.
git imap-send
git p4
Import from and submit to Perforce repositories.
git p4 clone //depot/path
git quiltimport
Applies a quilt patchset onto the current branch.
git quiltimport [patchdir]
git request-pull
Generates a summary of pending changes.
git request-pull [start] [url] [end]
git send-email
Send a collection of patches as emails.
git send-email *.patch
git svn
Bidirectional operation between Subversion and Git.
git svn clone [svn-repo-url]

Low-level Commands / Manipulators

git apply
Apply a patch to files and/or to the index.
git apply [patchfile]
git checkout-index
Copy files from the index to the working tree.
git checkout-index [file]
git commit-graph
Write and verify Git commit-graph files.
git commit-graph write --reachable
git commit-tree
Create a new commit object.
git commit-tree [tree] -m "Message"
git hash-object
Compute object ID and optionally create a blob from a file.
git hash-object -w [file]
git index-pack
Build pack index file for an existing packed archive.
git index-pack [pack-file]
git merge-file
Run a three-way file merge.
git merge-file local.txt base.txt remote.txt
git merge-index
Run a merge for files needing merging.
git merge-index git merge-one-file [files]
git mktag
Create a tag object with extra validation.
git mktag
git mktree
Build a tree object from ls-tree formatted text.
git mktree
git multi-pack-index
Write and verify multi-pack-indexes.
git multi-pack-index write
git pack-objects
Create a packed archive of objects.
git pack-objects [pack-name]
git prune-packed
Remove extra objects that are already in pack files.
git prune-packed
git read-tree
Reads tree information into the index.
git read-tree [tree-ish]
git symbolic-ref
Read, modify, and delete symbolic refs.
git symbolic-ref HEAD refs/heads/newbranch
git unpack-objects
Unpack objects from a packed archive.
git unpack-objects [pack-file]
git update-index
Register file contents in the working tree to the index.
git update-index --add [file]
git update-ref
Update the object name stored in a ref safely.
git update-ref refs/heads/branch [commit]
git write-tree
Create a tree object from the current index.
git write-tree

Low-level Commands / Interrogators

git cat-file
Provide content or type and size info for repository objects.
git cat-file -p [object]
git cherry
Find commits yet to be applied to upstream.
git cherry [upstream] [head]
git diff-files
Compare files in the working tree and the index.
git diff-files
git diff-index
Compare a tree to the working tree or index.
git diff-index HEAD
git diff-tree
Compare content and mode of blobs found via two tree objects.
git diff-tree HEAD~1 HEAD
git for-each-ref
Output information on each ref.
git for-each-ref
git for-each-repo
Run a Git command on a list of repositories.
git for-each-repo --config=repo.txt status
git get-tar-commit-id
Extract commit ID from an archive created using git archive.
git get-tar-commit-id < archive.tar
git ls-files
Show information about files in the index and the working tree.
git ls-files
git ls-remote
List references in a remote repository.
git ls-remote [remote]
git ls-tree
List the contents of a tree object.
git ls-tree HEAD
git merge-base
Find as many good common ancestors as possible for a merge.
git merge-base main feature
git name-rev
Find symbolic names for given revisions.
git name-rev [commit]
git pack-redundant
Find redundant pack files.
git pack-redundant --all
git rev-list
Lists commit objects in reverse chronological order.
git rev-list HEAD --
git rev-parse
Pick out and massage parameters.
git rev-parse HEAD
git show-index
Show packed archive index.
git show-index < packfile.idx
git show-ref
List references in a local repository.
git show-ref
git unpack-file
Create a temporary file with a blob’s contents.
git unpack-file [object]
git var
Show a Git logical variable.
git var GIT_AUTHOR_IDENT
git verify-pack
Validate packed Git archive files.
git verify-pack packfile.idx

# Internal script, generally not invoked directly.

Common Git shell script setup code (internal).
# Internal script, generally not invoked directly.
git stripspace

Additionally, Git provides documentation “topics” (not commands) such as:

  • attributes, cli, hooks, ignore, mailmap, modules, repository-layout, revisions
  • Developer-facing file format and protocol docs like format-bundle, format-chunk, format-pack, protocol-v2, etc.

These are reference documents rather than commands. Refer to git help [topic] for details.
